# Stack on link list

# Descripion

实现基于链表的栈  
之前实现过基于数组的栈结构。然而，使用数组有一个缺点，那就是数组的大小是一定的。 栈里元素达到一定数量时，入栈就会失败。
对于栈来说，每次的操作都是基于栈顶的。而单向链表的缺点是在获取元素的时候，需要沿着链表从头部开始搜索。  
如果使用一个单向链表来储存栈的数据，每一次的操作是对链表头部的操作。 

结构体 ``Stack`` 用来表示一个栈的抽象，包括：
1. ``size`` 栈的大小
2. ``top`` 指向栈顶的指针

结构体 ``Node`` 用来表示一个链表节点。  

# Hint
出栈就是删除内部链表的头节点， 入栈是相反的过程
