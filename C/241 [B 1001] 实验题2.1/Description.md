# [B 1001] 实验题2.1

# Description

编程计算下面数学表达式的值，其中a,b,c,x,y作为变量。
（即表达式如包括a, b, c, x 或 y, 从键盘输入相应的数值，程序输出表达式计算结果。）  
说明： 形如  m ^ n 表示 m的n 次幂。  
（1） 1200 / ( 24 - 4 \* 5 ) ;  
（2）c \* ( a^2 + b^2 )^( 1/2 ) ;  
（3）lnln( 10^3.5 + 2 ) ;  
（4）y + a % 5 * (int)(x + y) / 2 % 4 。  

# Input

输入只有一行，该行有5个数，它们之间都用一个空格隔开，
该行的前三个数是整数，分别对应 整型变量a、b、 c，后两个数是浮点数，分别对应 x和 y。

# Output

输出分4行，每行都是一个浮点数。所有的浮点数都要求保留三位小数。
4行数据分别对应：
表达式1的结果
表达式2的结果
表达式3的结果
表达式4的结果

# Sample Input

```
3 4 5 1 2
```

# Sample Output

```
300.000
25.000
2.087
2.000
```
