# 回文数

# Description
## 回文数是正着读和逆着读都相同的数，比如7,1221,12321都是回文数，而1234则不是回文数，现在给出一些数，请你判断它们是不是回文数

## input
第一行是一个整数 n，代表数字的个数，0 < n <= 100

第二行是n个整数，是需要你判断的数，每个数的范围是[0, 100000000]

## output
输出 n 行，每一行输出一个"Yes"或"No"，代表对应的数字是不是回文数

## sample input
```
3
7
2345
12321
```

## sample output
```
Yes
No
Yes
```
