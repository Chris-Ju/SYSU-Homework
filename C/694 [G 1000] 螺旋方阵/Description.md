# [G 1000] 螺旋方阵

# Description
请打印出螺旋方阵。如4\*4的矩阵效果是：
```
1 2 3 4
12 13 14 5
11 16 15 6
10 9 8 7
```

# Input

矩阵的维度n，如上例中n=4

# Output

螺旋方阵. <font style="color: red;"> **输出时同一行中，相邻两个数用一个空格隔开，行末无空格. 最后一行也需要输出'\n'.**</font>

# Sample Input

```
4
```

# Sample Output

```
1 2 3 4
12 13 14 5
11 16 15 6
10 9 8 7
```
